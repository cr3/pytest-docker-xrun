Requirements
------------

You will need the following prerequisites to use pytest-pytest-docker-xrun:

- Python 3.9, 3.10, 3.11, 3.12, 3.13

Installation
------------

To install pytest-pytest-docker-xrun:

.. code-block:: bash

  $ pip install pytest-pytest-docker-xrun
